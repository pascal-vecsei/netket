class netket.graph.Graph(edges, n_nodes=None)ÔÉÅ

Bases: netket.graph.AbstractGraph

A simple implementation of Graph based on an external graph library.

The underlying implemnetation is based on igraph and supports conversion to networkx, but this is an implementation detail and could be changed in the future.
