Creates a tensorboard logger using tensorboardX‚Äôs summarywriter. Refer to its documentation for further details

https://tensorboardx.readthedocs.io/en/latest/tensorboard.html

TensorBoardX must be installed.

Parameters

  • comment (string) ‚Äì Comment logdir suffix appended to the default logdir. If logdir is assigned, this argument has no effect.

  • max_queue (int) ‚Äì Size of the queue for pending events and summaries before one of the ‚Äòadd‚Äô calls forces a flush to disk. Default is ten items.
